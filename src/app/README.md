. Instrucțiuni de Pornire 
    Generare: Deschideți Generate_Training_Data.vi și apăsați butonul Run pentru a crea fișierul de date data_meteo.csv.

    Antrenare: Deschideți Antrenare_Retea.vi. Asigurați-vă că arhitectura este setată pe 3 neuroni ascunși și rulați VI-ul pentru a genera fișierul training.nn si fisierul training_model.nn.

    Utilizare: Deschideți Predictie.vi. Odată pornit, selectati o imagine cu exemple de nori din data/test/Cloud Samples mișcați sliderele de pe panou pentru a vedea cum rețeaua prezice producția de energie în funcție de scenariul meteo setat.
